# Rythm Game
## Is a VSRG (Vertical Scrolling Rythm Game) like Osu!mania or Quaver.
## I'm making this project because I love this kind of games and I want to build one, it doesn't solve any real problem but it's really fun and you improve your reaction time and finger control.

## My target users are teenagers.

## Planned features
- Storage of score
- Mods
- Size of notes
- Scroll speed
- Keybinds

## Program structure
There will be an home page, settings, and beatmaps.

## Challenges
I think the creation of the beatmaps will be a hard part for the creation of the game.
